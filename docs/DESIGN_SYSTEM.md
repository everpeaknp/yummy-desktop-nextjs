## Color Palette

### Primary Brand (Orange)
| Token | Hex | Usage |
|-------|-----|-------|
| `primary` | `#F24E1E` | Main branding, buttons, active states |
| `primaryDark` | `#D63D12` | Hover states, pressed states |
| `primarySoft` | `#FFF1E7` | Backgrounds for active items |
| `gradientStart` | `#FFFF6B2B` | Gradient start |
| `gradientEnd` | `#F24E1E` | Gradient end |

### Secondary Brand (Blue)
| Token | Hex | Usage |
|-------|-----|-------|
| `premiumBlue` | `#2563EB` | Global actions, links |
| `blueDark` | `#1E3A8A` | Deep contrast |
| `blueSoft` | `#EFF2FF` | Subtle blue backgrounds |

### Status Colors
| Status | Hex | Token |
|--------|-----|-------|
| Success | `#10B981` | `emeraldGreen` |
| Warning | `#F59E0B` | `statusWarning` |
| Error | `#EF4444` | `statusError` |
| Info | `#3B82F6` | `statusInfo` |

### Neutrals (Light Mode)
| Token | Hex | Usage |
|-------|-----|-------|

| `surfaceSubtle` | `#F9FAFB` | Inputs, secondary backgrounds |
| `border` | `#E5E7EB` | Borders, dividers |
| `textPrimary` | `#111827` | Main text |

### Neutrals (Dark Mode)

## UI Components
- **Input Fields**: Rounded corners (12px), Filled (Light Grey), Outlined on Focus (Primary Orange).
- **Cards**: Large Radius (20px), Shadow (Light), No Elevation (Dark).
- **Buttons**: Rounded Medium (12px), Primary Orange Background, White Text.
